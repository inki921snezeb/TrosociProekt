# Калкулатор за животни трошоци

Овој проект е калкулатор за животни трошоци кој помага на корисниците да го следат и управуваат својот месечен буџет. Апликацијата ќе овозможи внесување на разни трошоци (стан, храна, превоз, и други) и автоматски ќе пресмета вкупен месечен износ.
