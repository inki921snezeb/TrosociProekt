
Калкулатор за трошоци за живот
Опис
Овој проект е веб-апликација за пресметка и следење на месечните трошоци за живот. Корисниците можат да додаваат категории на трошоци и да ги следат своите финансии.

Карактеристики

Внесување трошоци.
Пресметка на вкупни трошоци.
Преглед на статистики.
Технологии
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express
База на податоци: MongoDB
